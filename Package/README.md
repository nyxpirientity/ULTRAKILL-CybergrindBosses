Cybergrind Bosses is a mod which as the name suggests adds as cheat which when enabled, adds (currently) most of the bosses from ULTRAKILL to The Cyber Grind. It features *a lot* of options for configuring the balance of when bosses spawn, the amount of points they'll cost for the wave, the frequency they spawn, etc. Or you can just turn off each individual boss if you please.

## design note
I did take some creative liberties with certain aspects of this mod in order to fit things into the cybergrind, as most bosses are, shockingly, not designed to be fought outside of their respective levels. This includes some having mechanics not usually seen, or featuring mechanics to handle them dropping off of the edge.

## included Bosses
The goal is to feature as many bosses from the game as possible, as given the optional and configurable nature of the mod, players should be able to remove any they rather not have.
The bosses in the mod currently are:
- Swordsmachines Tundra and Agony
- V2
- V2 Second
- Gabriel
- Gabriel Second
- Cancerous Rodent
- Very Cancerous Rodent
- 'Mandalore' (or whatever the full auto boss was named in-game)
- Blood/Suicide Trees (features behaviour/mechanics not seen in the base game)
- Leviathan
- (Corpse of King) Minos
- Minotaur
- Flesh Prison
- Flesh Panopticon
- Minos Prime
- Sisyphus Prime
- Earthmover/Centaur Defence System (features behaviour/mechanics not seen in the base game)
- Geryon

## configurability
As stated before, the mod is meant to be highly configurable, as I balanced it with myself as the sole play-tester and designed any mechanics I needed to, also with myself as the sole play-tester. Most options should have a description, but tbh I may have forgotton some that need one. Options are configured on a per-boss basis, and some non-bosses have *attributes* which are currently used just for configuring how enemies are handled when a fake fall begins. Options function through BepInEx config files, so whether they show in menus depend on if the mod you're using to configure mods through in-game menus automatically supports mods using BepInEx config files. It also automatically reloads the config from the file every time the window regains focus, so you can just use a text editor. (make sure you save!)
